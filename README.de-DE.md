### Erste Schritte

Bevor Sie dieses Repository klonen und ausfÃ¼hren, mÃ¼ssen Sie [Git](https://git-scm.com), [Node.js](https://nodejs.org/en/download/) (mit [npm](https://www.npmjs.com/)) und [pnpm](https://pnpm.io/) auf Ihrem Computer installiert haben. Kommandozeilenaufrufe:

```shell

# Run the web

# Run the server

# Run the desktop

# Run the web
pnpm run start:web
# Run the desktop

# Build the web
pnpm run build:web
# Build the desktop
pnpm run build:desktop
# Clear node_modules
pnpm run clear
```
